# Tabelog TUI - 使用指南

## 啟動 TUI

使用以下命令啟動互動式 TUI 介面:

```bash
# 使用 uv
uv run tabelog tui

# 或直接使用 Python
python -m tabelog.tui
```

## 介面說明

TUI 介面分為四個主要區域:

### 1. 標題列 (Header)
- 顯示應用程式名稱和狀態

### 2. 搜尋面板 (Search Panel)
- **地區輸入框**: 輸入要搜尋的地區 (例如: 東京、大阪)
- **關鍵字輸入框**: 輸入搜尋關鍵字 (例如: 寿司、ラーメン)
- **排序方式下拉選單**: 選擇排序方式
  - 評分排名 (預設)
  - 評論數排序
  - 新開幕
  - 標準排序
- **搜尋按鈕**: 點擊或按 Enter 執行搜尋

### 3. 結果列表 (Results Table)
- 顯示搜尋結果的餐廳列表
- 欄位包括:
  - 名稱
  - 評分
  - 評論數
  - 地區
  - 類型

### 4. 詳細資訊面板 (Detail Panel)
- 顯示選中餐廳的詳細資訊
- 包括:
  - 餐廳名稱
  - 評分和評論數
  - 地區和車站資訊
  - 料理類型
  - 價格範圍
  - 餐廳 URL

### 5. 底部列 (Footer)
- 顯示可用的鍵盤快捷鍵

## 鍵盤快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `q` | 退出應用程式 |
| `s` | 聚焦到搜尋輸入框 |
| `r` | 聚焦到結果列表 |
| `d` | 聚焦到詳細資訊面板 |
| `↑` / `↓` | 在結果列表中上下移動 |
| `Tab` | 在元件間切換 |
| `Enter` | 在搜尋框中執行搜尋 |

## 使用流程

1. 啟動 TUI: `uv run tabelog tui`
2. 在「地區」輸入框中輸入地區名稱 (例如: 東京)
3. 在「關鍵字」輸入框中輸入關鍵字 (例如: 寿司)
4. 使用 `Tab` 鍵移到「排序方式」下拉選單
5. 用 `↑` / `↓` 選擇排序方式，按 `Enter` 確認
6. 點擊「搜尋」按鈕或在輸入框按 `Enter`
7. 等待搜尋完成,結果會顯示在結果列表中
8. 使用方向鍵或點擊選擇餐廳
9. 詳細資訊會自動顯示在詳細資訊面板中
10. 按 `q` 退出應用程式

## 範例搜尋

### 搜尋東京的壽司店（評分排名）
- 地區: `東京`
- 關鍵字: `寿司`
- 排序: `評分排名`

### 搜尋大阪的拉麵店（評論數排序）
- 地區: `大阪`
- 關鍵字: `ラーメン`
- 排序: `評論數排序`

### 搜尋京都的和食（新開幕）
- 地區: `京都`
- 關鍵字: `和食`
- 排序: `新開幕`

## 注意事項

- 搜尋可能需要幾秒鐘時間,請耐心等待
- 確保有網路連線
- 搜尋結果依賴 Tabelog 網站的可用性
- 建議使用較大的終端機視窗以獲得最佳體驗
